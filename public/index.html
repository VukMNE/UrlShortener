URL Shortener

This microservice (called micro, because of its simplicity) servers as a mechanism for shortening and saving shortened versions of url.

Insert of urls is possible using the form above, or, directly from the address bar. Once the url is inserted, the microservice will respond with json object telling the user what is the shortened version of the url, which if used, will redirect to original url.
